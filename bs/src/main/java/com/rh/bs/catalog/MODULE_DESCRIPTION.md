# Módulo de Catálogo (Catalog)

## Descripción
Este módulo es la fuente de verdad para toda la información descriptiva de los productos que la empresa ofrece. Es un módulo mayoritariamente de lectura intensiva.

## Responsabilidades
*   **Gestión de Productos:** Crear, editar y listar productos con sus detalles (nombre, descripción, precios base, especificaciones).
*   **Categorización:** Administración de la jerarquía de categorías (`PRODUCT_CATEGORY`).
*   **Marcas:** Gestión de fabricantes o marcas (`BRAND`).
*   **Multimedia:** Asociación de imágenes y recursos visuales a los productos.
*   **Estados:** Control del ciclo de vida del producto (Borrador, Activo, Descontinuado).

## Datos Clave
*   Productos y sus atributos.
*   Categorías.
*   Marcas.
*   Histórico de Precios Base.
