# Módulo de Experiencia de Cliente (Customer Experience)

## Descripción
Este módulo agrupa funcionalides orientadas a aumentar el compromiso (engagement), la fidelización y la interacción del usuario con la plataforma, más allá de la transacción de compra pura.

## Responsabilidades
*   **Cupones y Descuentos:** Motor de reglas para la creación, validación y aplicación de cupones promocionales (`COUPONS`, `COUPON_TYPES`).
*   **Listas de Deseos (Wishlist):** Permitir a los usuarios guardar productos para comprar después.
*   **Reseñas y Calificaciones:** Gestión del feedback de los clientes sobre los productos (`PRODUCT_CUSTOMER_REVIEW`).

## Datos Clave
*   Cupones.
*   Reglas de negocio para descuentos.
*   Listas de deseos de usuarios.
*   Comentarios y ratings.
